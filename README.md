# Thước Lỗ Ban Online — Nội Thất The One

Công cụ tra cứu **thước lỗ ban** chuẩn phong thủy cho website [Nội Thất The One](https://noithattheone.vn/).
Người dùng nhập kích thước (mm / cm / m), công cụ cho biết kích thước rơi vào **cung tốt (đỏ)** hay **cung xấu (đen)**, kèm tên cung lớn, phân cung, ý nghĩa và **gợi ý kích thước đẹp gần nhất**.

## Tính năng

- Hỗ trợ **3 loại thước**:
  - **52.2cm — Thông thủy**: cửa đi, cửa sổ, cổng (khoảng rỗng/lọt lòng)
  - **42.9cm — Dương trạch**: bệ bếp, bậc thang, khối xây dựng đặc
  - **38.8cm — Nội thất / Âm phần**: bàn, tủ, giường, kệ, bàn thờ
- Thước trực quan tô màu đỏ/đen, có con trỏ chỉ vị trí số đo.
- Tự gợi ý kích thước đẹp gần nhất khi kết quả là cung xấu.
- Đổi đơn vị mm / cm / m, giữ nguyên kích thước thực.
- **1 file `index.html` độc lập** — không phụ thuộc thư viện, dễ nhúng vào website.

## Cách dùng / Tích hợp

Mở `index.html` bằng trình duyệt, hoặc xem bản demo qua GitHub Pages.

Để nhúng vào noithattheone.vn:
- **Cách đơn giản:** nhúng qua iframe trỏ tới link GitHub Pages.
- **Cách liền mạch:** copy nội dung `index.html` vào một trang/landing page của website.

## Dữ liệu

Dữ liệu cung & phân cung tổng hợp từ các nguồn thước lỗ ban phổ biến và thống nhất giữa nhiều nguồn. Màu sắc (tốt/xấu) xác định theo cung lớn. Công cụ mang tính tham khảo phong thủy.
